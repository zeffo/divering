# `divering`

A [ringfairy](https://github.com/k3rs3d/ringfairy) powered webring for divers of all kinds.

To join:
1. Add the following anchor tags somewhere on your site:
```html
<a href="https://dive.zeffo.me/{SLUG}/previous">←</a>
<a href="https://dive.zeffo.me/">divering</a>
<a href="https://dive.zeffo.me/{SLUG}/next">→</a>
```
2. Add your website to `websites.json` and open a pull request.
